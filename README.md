# BIM LiDAR Scanner

Нативное iOS приложение для сканирования с LiDAR → отправка на ПК в BIM Scanner.

## Установка (бесплатно, без $99)

### Шаг 1: Загрузить IPA с GitHub

1. Создайте аккаунт на github.com (бесплатно)
2. Создайте новый репозиторий, загрузите все файлы этой папки
3. Перейдите в Actions → Build iOS IPA → Run workflow
4. Когда сборка завершится — скачайте `BIMLidar-IPA` из Artifacts

### Шаг 2: Установить AltStore на ПК

1. Скачайте AltServer с altstore.io
2. Установите iTunes (если нет)
3. Запустите AltServer — он появится в трее

### Шаг 3: Установить AltStore на iPhone

1. Подключите iPhone к ПК по USB
2. В трее AltServer → Install AltStore → выберите ваш iPhone
3. Введите Apple ID когда попросит (бесплатный аккаунт)
4. На iPhone: Настройки → Основные → VPN и управление → Доверять

### Шаг 4: Установить BIM LiDAR через AltStore

1. Откройте AltStore на iPhone
2. Нажмите + → выберите скачанный BIMLidar.ipa
3. Приложение установится

### Обновление каждые 7 дней

AltStore обновляет автоматически когда iPhone в той же Wi-Fi что и ПК с AltServer.

## Использование

1. Запустите BIM Scanner на ПК: `python bim_scanner.py`
2. В настройках приложения укажите IP компьютера (написан в консоли)
3. Нажмите "Начать сканирование" — медленно обходите объект
4. Нажмите "Остановить" когда достаточно точек
5. Нажмите "Отправить на ПК" — скан появится в BIM Scanner
6. Конвертируйте в DXF/OBJ/GLTF

## Требования

- iPhone 12 Pro / 13 / 14 / 15 (любой с LiDAR) или iPad Pro
- iOS 16+
